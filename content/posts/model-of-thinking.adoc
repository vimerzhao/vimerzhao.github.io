= 信息、思考、知识

最近在思考的一个问题，当一件热点事件发生的时候，如何能够透过各种声音看到问题的本质，并给出客观、全面的评价，这其实是一件非常困难的事情，我觉得必须具备三要素。

== 知识

知识就像机器学习里面的训练集。放到具体生活中，就是我们的阅历、经验。这是非常重要的，但是目前信息碎片化严重，很少有人能够长时间、有深度地去阅读、思考，很多优秀的书籍被束之于高阁。 *目前最紧要的就是阅读、积淀高质量的知识* ，腹有诗书气自华，这绝对不是网上学两个烂梗能达到的水平。知识积淀的一个重要里程碑就是拥有一个庞大而结构合理的知识体系。

== 信息

信息就像机器学习里面的测试集。信息，必须是全面、客观、真实的信息，但目前的环境是标题党、煽情文恣意横行，想要获得真正有价值的信息其实也不太容易，但相比知识的积淀还是要容易很多，只要掌握一些工具、技巧和套路，都是可以获得高质量信息的。

== 思考

思考就像机器学习里面的算法。思考可以从知识中学习、提炼，但更多的需要自己去完善、打磨，每个独立的个体都要形成自己的世界观和方法论，思考不应该是很虚的，相反，思考应该是有方法可遵循、有模型可套用，然后在此基础上，迸发出灵感，体现出差异性。

以上，一次简单的头脑风暴。
